# 🛒 Haskell Shopping List Manager

## 📋 Project Overview
A functional programming project for PRG2214: Functional Programming Principles. This is a console-based shopping list manager implemented in Haskell, demonstrating core FP concepts.

## 🎯 Learning Objectives Demonstrated

### ✅ Functional Programming Concepts
1. **Pure Functions**: All list operations are pure
2. **Immutability**: No in-place modifications
3. **Higher-Order Functions**: `map`, `filter`, `foldr`
4. **Recursion**: Menu system uses tail recursion
5. **Type System**: Strong, static typing with custom types
6. **Polymorphism**: Category and Priority types
7. **Typeclasses**: `Printable` instance
8. **Monadic I/O**: File operations using IO monad
9. **Pattern Matching**: Extensive use in menu handling
